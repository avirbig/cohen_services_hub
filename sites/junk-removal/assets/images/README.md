# Images Directory

Place your images here:

- Hero images
- Service icons
- Logo
- Background images
- Any photos for the site

## Recommended formats:
- **JPG/JPEG** - For photographs
- **PNG** - For graphics with transparency
- **SVG** - For icons and logos (scalable)
- **WebP** - For optimized web images

## Naming convention:
- Use lowercase
- Use hyphens instead of spaces
- Be descriptive: `hero-background.jpg`, `service-furniture.png`
